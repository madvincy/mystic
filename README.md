# Mystic Wines & Spirits 🍷

A modern e-commerce platform for premium wines and spirits.

## 🚀 Features

- ✅ Next.js 14 with App Router
- ✅ TypeScript
- ✅ Tailwind CSS + shadcn/ui
- ✅ Supabase Authentication & Database
- ✅ Google OAuth
- ✅ M-Pesa Payments Integration
- ✅ IndexedDB + Redux for state management
- ✅ Wishlist, Discounts, Flash Sales
- ✅ Admin Dashboard
- ✅ Dark/Light Theme
- ✅ Responsive Design

npm install
